# Toil Tracker — Issue Tracker

This is the **public issue tracker** for Toil Tracker (Work Audit) — a facilitated team tool for quarterly toil-tracking retrospectives.

The app itself is closed source, but issue reporting is open. If you found a bug, have a suggestion, or want to share feedback, you're in the right place.

---

## Reporting an Issue

Click **[New Issue](https://github.com/sargismarkosyan/toil-tracker-issues/issues/new)** and include:

- **What you were doing** when the problem occurred
- **What you expected** to happen
- **What actually happened**
- **Steps to reproduce** (if it's a bug)
- Your browser and device (e.g. Chrome on macOS, Safari on iPhone)

Screenshots are always helpful.

---

## Feature Requests

Have an idea? Open an issue and prefix the title with `[Feature]` — e.g. `[Feature] Export to CSV`.

---

## What This Repo Is Not

- It does not contain the application source code
- Pull requests here will not be accepted (there's no code to change)
- For security-sensitive issues, please email directly rather than opening a public issue
